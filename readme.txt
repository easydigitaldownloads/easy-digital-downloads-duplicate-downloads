=== Easy Digital Downloads Duplicate Downloads ===


== Description ==


This plugin allows to duplicate an exisiting EDD product




Option 1:

In 'Edit Products', you can click on 'Duplicate Me!' next to the Update button on the right.



Option 2:

In 'All Products' on mouseover of a product you can click on "Duplicate"
